# Brodigy Community Platform

A modern, responsive membership platform designed for **Brodigy**—an exclusive brotherhood of ambitious, purpose-driven men committed to leadership, personal growth, networking, and meaningful community.

Built on the **HTML5** framework and extensively customized to provide a premium user experience for member registration, community engagement, and member management.

---

## Overview

Brodigy is more than a website—it's a digital platform that connects members through brotherhood, mentorship, leadership, and shared opportunities.

The platform provides a seamless experience for prospective and existing members through a modern horizontal scrolling interface, responsive design, and intuitive navigation.

---

### Public Website


Navigate into the project:

```bash

```

Open `index.html` in your browser or run the project using your preferred local server.
